// 剧集管理功能特性：
- 按剧目分组显示剧集
- 支持搜索剧集名称和剧目名称
- 可按剧目筛选剧集
- 显示剧集时长、状态等信息
- 支持创建、编辑、删除剧集
- 弹窗式创建界面，简化操作流程

// 数据库字段映射：
- titleId: 关联的影片ID
- epNumber: 集数编号
- name: 集数名称（可选）
- videoId: 视频文件ID或URL
- durationSec: 时长（秒）
- status: 发布状态（DRAFT/PUBLISHED）
